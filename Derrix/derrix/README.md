## Le nom de mon projet est > Derrix

Système Intéligent de Gestion des Contacts

##### ---------------------------------------------------------------

## Description du Projet

Derrix est une application web simple développée avec Laravel permettant aux utilisateurs d’envoyer des messages via un formulaire de contact.
Les messages sont stockés dans une base de données et affichés dans une interface d’administration.

Ce projet met l’accent sur la simplicité, la clarté et une structure professionnelle adaptée à une présentation académique.

##### ---------------------------------------------------------------


## Côté Utilisateur

    Formulaire de contact (nom, e-mail, message)

    Système d'envoi de messages

    Message de confirmation de réussite

##### ---------------------------------------------------------------

## Côté Administrateur

    Visualisation de tous les messages reçus

    Marquer les messages comme lus/non lus

    Supprimer des messages

    Interface de tableau de bord simple

##### ---------------------------------------------------------------

## Technologies Utilisées

Laravel (Framework PHP)
PHP
MySQL (Base de données)
Bootstrap (Design UI simple et responsive)
Laravel Mail (envoi d’emails)

##### ---------------------------------------------------------------

## Guide d'Installation

Cloner le projet :
git clone <repo>
Installer les dépendances :
composer install
Configurer le fichier .env :
Connexion MySQL
Configuration Mail
Lancer les migrations :
php artisan migrate
Démarrer le serveur :
php artisan serve
